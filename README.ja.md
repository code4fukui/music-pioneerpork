# music-pioneerpork

放牧和豚（パイオニアポーク）の曲を再生できるミュージックプレイヤーアプリです。

## 機能
- 放牧和豚の曲を再生できる
- ランダムな再生順序でプレイリストを再生
- お気に入りの曲を優先して再生 

## データ・API
曲データは[放牧和豚プレイリスト](https://suno.com/playlist/784c5dea-07fb-405b-b47b-b39f38b401ff)から取得しています。

- [music-opendata-fukui](https://github.com/code4fukui/music-opendata-fukui)
