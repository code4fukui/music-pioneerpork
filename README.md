# music-pioneerpork

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Music player app for the "Pioneer Pork" songs.

## Demo
[music-pioneerpork](https://code4fukui.github.io/music-pioneerpork/)

## Features
- Plays songs from the "Pioneer Pork" playlist
- Shuffles the playlist
- Supports Media Session API for playback controls

## Data / API
The songs and metadata are sourced from the [music-opendata-fukui](https://github.com/code4fukui/music-opendata-fukui) project.
